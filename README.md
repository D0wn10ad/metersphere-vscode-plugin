# MeterSphere VSCode Extension (Phase 1 MVP1 + Phase 2 Navigator)

This repository contains a VSCode extension plan and initial scaffold for MeterSphere integration targeting v2.x semantics. Phase 1 MVP1 implements token-based authentication, workspace/project persistence, and a WebView-based API debugger with on-demand sync. Phase 2 introduces a Core Navigator (TreeView) and full Option C command-driven UX with on-demand sync.

Note: This repository currently includes scaffolding and plan-level artifacts. The patch introduces minimal runnable TypeScript scaffolding with a WebView-based debugger; the full Phase 2 Navigator integration will be completed in subsequent commits following Phase 1 validation.
